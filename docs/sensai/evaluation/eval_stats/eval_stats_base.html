Collects data for the evaluation of predicted labels (including multi-dimensional predictions) and computes corresponding metrics

__init__(y_predicted: Union[numpy.ndarray, pandas.core.series.Series, pandas.core.frame.DataFrame, list], y_true: Union[numpy.ndarray, pandas.core.series.Series, pandas.core.frame.DataFrame, list], metrics: List[TMetric], additionalMetrics: List[TMetric] = None)¶
Parameters

  • y_predicted – sequence of predicted labels. In case of multi-dimensional predictions, a data frame with one column per dimension should be passed

  • y_true – sequence of ground truth labels of same shape as y_predicted

  • metrics – list of metrics to be computed on the provided data

  • additionalMetrics – the metrics to additionally compute. This should only be provided if metrics is None
