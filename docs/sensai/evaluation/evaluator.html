Constructs an evaluator with test and training data. Exactly one of the parameters {testFraction, testData, } must be given

Parameters

  • data – the full data set, or, if testData is given, the training data

  • testData – the data to use for testing/evaluation; if None, must specify either dataSplitter testFraction or dataSplitter

  • dataSplitter – [if testData is None] a splitter to use in order to obtain; if None, must specify either testData or testFraction

  • testFraction – [if testData is None, dataSplitter is None] the fraction of the data to use for testing/evaluation; if None, must specify either testData or dataSplitter

  • randomSeed – [if data is None, dataSplitter is None] the random seed to use for the fractional split of the data

  • shuffle – [if data is None, dataSplitter is None] whether to randomly (based on randomSeed) shuffle the dataset before splitting it
